# Alumioro

Site institucional estático para a Alumioro, serralheria de esquadrias de alumínio sob medida.

## Estrutura

- `index.html`: página principal.
- `demonstracao.html`: página de portfólio com exemplos de modelos oferecidos.
- `styles.css`: estilos responsivos.
- `script.js`: menu mobile.
- `assets/`: imagens publicaveis do site.
- `arquivos-originais/`: materiais base locais, ignorados pelo Git.
